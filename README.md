# GotSchedule: Online Schedule Maker App

**Your Personal Time Management Tool**

Welcome to **GotSchedule**, an innovative online scheduling app I developed as my final project for my Web Systems and Technologies (Websys) course. I designed this powerful tool to help you effectively manage your time, organize your tasks, and create personalized schedules. Whether you're a student, a professional, or simply someone looking to streamline your daily routine, GotSchedule offers a user-friendly interface and customizable features tailored to meet your unique needs.

---

## 🚀 System Overview & Architecture

GotSchedule is a **High-Performance Productivity & Scheduling Platform** designed to transform "chaotic to-dos" into "clear action plans." The system utilizes a modern **"Light Glass"** design language for maximum readability and a professional, airy feel.

### Core Value Proposition
GotSchedule combines task management, long-term calendar visualization, and an intelligent "Magic Scheduler" to give users **Time Sovereignty**. It moves beyond a simple list to offer a unified "Command Center" for personal and collaborative productivity.

---

## 🛠️ System Modules & Features

### 1. The Entrance Hub (`main.php`)
* **Landing Page:** A high-impact "Front Door" featuring a Hero section with dashboard previews, feature pillars, and trust signals.
* **Explanatory Purpose:** Clearly defines the "How-To" of the system (Three Superpowers: Scheduler, Notes, Grid).

### 2. The Pro Dashboard (`task.php`)
* **Central Task List:** Status-aware task tracking (Pending, In-Progress, Completed).
* **Mini-Calendar Widget:** A quick-access date picker for immediate contextual awareness.
* **Smart Stat Cards:** Real-time visualization of current workload.

### 3. The Magic Scheduler (`sched_maker.php`)
* **Dynamic Timeline:** Automatically converts your task list into a clear, time-blocked daily flow.
* **High-Performance Plan:** Focuses on "Acceleration" by breaking large goals into actionable, timed chunks.

### 4. Unified Notes (`notes.php`)
* **Idea Capture:** A dedicated glass-grid for capturing thoughts, inspirations, and research.
* **Color Tagging:** Visual organization for quick mental categorization.

### 5. Visual Calendar Grid (`calendar.php`)
* **Long-term Mapping:** A synchronized 7-column grid view for managing monthly milestones.
* **Full Transparency:** See exactly how your month is balanced across different categories.

### 6. Admin & Collaboration Hub (`admin.php`, `messages.php`)
* **Role-Based Access:** Specialized views for Administrators and Advisers.
* **System Analytics:** Real-time monitoring of user activity and organizational health.
* **Real-time Messaging:** Direct collaboration channel for advisers and students.

## 🎨 Design System: "Light Glass"

The system utilizes a custom-engineered UI/UX language:
* **High Contrast:** Dark-slate text on clear, semi-transparent backgrounds for zero eye strain.
* **Slate-Ink Boundaries:** Soft borders (`rgba(30, 41, 59, 0.12)`) replacing harsh lines.
* **Dynamic Elevation:** Interactive hover states and "Scale & Slide" transitions for a life-like feel.
* **Performance Optimized:** Carefully balanced blurs (12px) and animations to ensure zero lag on any device.

## 💻 Technical Stack

* **Backend:** PHP 8.2.12 (XAMPP environment)

* **Typography:** Inter (900/700/600/400 weights) for premium readability
* **Icons:** Bootstrap Icons 1.11.3
