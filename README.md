# GotSchedule: High-Performance Productivity Ecosystem 🚀

**Master Your Moment. Conquer Your Day.**

GotSchedule is a professional-grade, multi-dimensional productivity platform built to transform academic and professional chaos into streamlined action plans.
-https://gotschedule.ct.ws/
---

## 🛠️ Technical Stack (The Engine)
*   **Backend:** PHP 8+ (Optimized for XAMPP/Apache environments)
*   **Database:** MySQL (Relational schema with audit-aware architecture)
*   **Frontend Design:** 
    *   **Vanilla CSS3:** Custom design tokens for a "Light Glass" and "Midnight Glass" aesthetic.
    *   **Bootstrap 5.3.3:** Structural integrity and utility components.
    *   **Inter Font System:** Variable weighted typography (400-900) for maximum clarity.
*   **Interactions:** AJAX/Fetch API for a "Single Page App" feel.

---

## 💎 Premium Features

### 📅 Advanced Scheduling & Calendar
*   **Dynamic Recurrence Engine**: Project Daily, Weekly, or Monthly tasks dynamically.
*   **Time-Slot Validation**: Intelligent overlap prevention for scheduling.
*   **Visual Grid**: High-precision 7-column interactive calendar with "Click-to-Create".
*   **Smart Export**: Sync via the **ICS Export** system.

### 🧠 Cognitive Performance Tools (The Dashboard)
*   **The Brain Dump Buffer**: A dedicated 24-hour scratchpad that auto-clears.
*   **Habit Tracker Grid**: A visual 31-day streak tracker.
*   **Countdown Timers**: Real-time "Days to Deadline" calculations.

### 🤝 Strategic Collaboration Hub
*   **Secure Task Tagging**: Tag teammates in tasks using exact-username search.
*   **Shared Notes & Collab Keys**: Unique keys to share specific notes with peers. 
*   **Collaboration Portal**: Dedicated hub for shared resources.

### 🌓 Midnight Mode (The Theme Engine)
*   **Persistent Theming**: State-aware system (Light/Dark).
*   **Midnight Glass UI**: Carefully tuned dark mode typography and surfaces.

### 🛡️ System Reliability & Security
*   **Audit Logging**: Detailed tracking of all major actions.
*   **Session Lockdown**: Strict authentication checks prevent URL-direct access.
*   **Soft Delete**: Data safety via `deleted_at` timestamps.

---

## 🚀 Installation & Setup
1. Clone the project into your `xampp/htdocs/gotschedule` directory.
2. Import `database.sql` into your MySQL server.
3. Update `config.php` with your local DB credentials.
4. Access via `http://localhost/gotschedule/main.php`.
